## Installation

```ruby
gem 'user_submitted'
```

And then execute:
```bash
$ bundle
```
Add all the Gem migration files with:
  bundle exec rake user_submitted_engine:install:migrations

Or install it yourself as:
```bash
$ gem install user_submitted
```
